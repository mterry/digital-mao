# User Journey Maps - Digital Mao

## Journey Map 1: New Player's First Game Experience

### Persona: Jamie Park (Curious Newcomer)

### Journey Overview
Jamie discovers Digital Mao through a friend's recommendation and plays their first game, experiencing the unique learning-through-failure mechanic.

---

### Stage 1: Discovery & Entry (0-2 minutes)

**Actions:**
- Clicks link shared by friend
- Lands on game homepage
- Reads brief description
- Clicks "Join Game" button
- Sees list of available games

**Thoughts:**
- *"This sounds interesting but confusing"*
- *"I hope I don't embarrass myself"*
- *"Which game should I join?"*

**Emotions:**
- 😟 Curious but anxious
- 🤔 Uncertain about what to expect

**Pain Points:**
- No clear indication of beginner-friendly games
- Unclear what "hidden rules" means
- Worried about joining experienced players

**Opportunities:**
- Add "New Player Friendly" badge to games
- Provide quick tutorial option
- Show player experience levels

**Touchpoints:**
- Homepage
- Game browser
- Join game button

---

### Stage 2: First Moments in Game (2-5 minutes)

**Actions:**
- Joins game with 3 other players
- Sees cards dealt to hand
- Reads chat messages from other players
- Waits for turn
- Observes first few plays

**Thoughts:**
- *"Okay, this looks like a normal card game"*
- *"I'll just watch what others do first"*
- *"That player just got penalized... why?"*

**Emotions:**
- 😰 Nervous about first turn
- 👀 Observing carefully
- 🤨 Confused by penalties

**Pain Points:**
- No explanation of why penalties occur
- Unclear when it's their turn
- Don't know what cards are valid to play

**Opportunities:**
- Highlight current player clearly
- Show basic matching rules (suit/rank)
- Provide gentle first-turn guidance

**Touchpoints:**
- Game board
- Hand display
- Chat window
- Turn indicator

---

### Stage 3: First Play & Penalty (5-8 minutes)

**Actions:**
- Turn arrives
- Plays a matching card
- Receives penalty: "Silence is golden"
- Draws 2 penalty cards
- Confused, reads penalty message again

**Thoughts:**
- *"What did I do wrong?"*
- *"I played a valid card..."*
- *"Maybe I was supposed to say something?"*
- *"Let me watch more carefully"*

**Emotions:**
- 😣 Frustrated by unclear feedback
- 🤔 Analyzing the situation
- 😤 Slightly annoyed but intrigued

**Pain Points:**
- Penalty feels unfair without explanation
- Cryptic message is too vague
- No way to verify understanding
- Hand size increased, harder to win

**Opportunities:**
- Make penalty messages slightly more helpful
- Provide penalty history for review
- Show pattern after multiple violations
- Celebrate first penalty as learning moment

**Touchpoints:**
- Penalty notification
- Judge message in chat
- Updated hand display
- Penalty history (if available)

---

### Stage 4: Pattern Recognition (8-15 minutes)

**Actions:**
- Plays several more turns
- Tries saying "Mao" before playing
- No penalty this time!
- Observes others getting penalized
- Takes mental notes of patterns

**Thoughts:**
- *"Aha! I need to say something when I play"*
- *"But what exactly triggers penalties?"*
- *"That player said something different..."*
- *"I'm starting to get this!"*

**Emotions:**
- 😊 Excited by discovery
- 🎯 Focused on learning
- 💡 Feeling clever

**Pain Points:**
- Still uncertain about all the rules
- Worried about new penalties
- Difficult to track all patterns

**Opportunities:**
- Provide subtle positive feedback for correct plays
- Show discovered rules after consistent compliance
- Celebrate learning milestones

**Touchpoints:**
- Successful card plays
- Chat observations
- Mental rule tracking
- Pattern recognition

---

### Stage 5: Growing Confidence (15-25 minutes)

**Actions:**
- Plays multiple turns without penalties
- Experiments with different card types
- Engages in chat with other players
- Asks cryptic questions to test theories
- Starts to enjoy the mystery

**Thoughts:**
- *"I think I understand the basic rules now"*
- *"There are probably more rules I haven't discovered"*
- *"This is actually really fun!"*
- *"I want to win so I can create a rule"*

**Emotions:**
- 😄 Enjoying the challenge
- 🎮 Engaged in gameplay
- 🤝 Connected with other players

**Pain Points:**
- Still discovering new rules unexpectedly
- Occasional penalties for unknown violations
- Uncertainty about special card effects

**Opportunities:**
- Provide progress indicators
- Show rule discovery achievements
- Enable social learning through chat
- Reward consistent correct play

**Touchpoints:**
- Smooth gameplay flow
- Chat interactions
- Rule discovery moments
- Penalty-free turns

---

### Stage 6: First Game Completion (25-35 minutes)

**Actions:**
- Doesn't win but finishes game
- Reviews penalty history
- Sees winner create new rule
- Decides to play another game
- Feels more confident

**Thoughts:**
- *"I didn't win but I learned a lot"*
- *"I want to try again with what I know"*
- *"That new rule sounds interesting"*
- *"Next time I'll do better"*

**Emotions:**
- 😌 Satisfied with learning
- 🎯 Motivated to improve
- 🔄 Ready to play again

**Pain Points:**
- Didn't get to create a rule
- Still don't know all the rules
- Want to validate understanding

**Opportunities:**
- Provide game summary with insights
- Show learning progress
- Suggest next steps for improvement
- Offer to play with same group

**Touchpoints:**
- Game end screen
- Statistics display
- Penalty history review
- Play again button

---

### Overall Journey Metrics

**Duration:** 35 minutes  
**Emotional Arc:** Anxious → Frustrated → Curious → Excited → Satisfied  
**Key Moments:**
1. First penalty (critical learning moment)
2. First successful play after discovery
3. Watching winner create rule

**Success Indicators:**
- Completes first game
- Discovers at least 2 rules
- Wants to play again
- Understands basic mechanics

## Cross-Journey Insights

### Common Patterns
1. **Confusion → Discovery → Mastery** arc appears in all journeys
2. **Observation** is critical learning mechanism
3. **Experimentation** drives engagement
4. **Social learning** through watching others
5. **Incremental understanding** builds confidence

### Critical Touchpoints
1. **First penalty** - Sets tone for learning
2. **First discovery** - Validates approach
3. **First success** - Builds confidence
4. **Mastery moment** - Creates satisfaction

### Emotional Peaks & Valleys
- **Valleys:** Initial confusion, unexpected penalties, unclear feedback
- **Peaks:** Discovery moments, successful tests, mastery achievement

### Design Implications

#### For Newcomers
- Gentle first penalties
- Clear visual feedback
- Observation opportunities
- Pattern recognition support

#### For Rule Creators
- Expressive input system
- Clear compilation feedback
- Testing capabilities
- Impact visibility

#### For All Users
- Consistent enforcement
- Fair penalty system
- Social learning support
- Progress tracking
- Achievement recognition
