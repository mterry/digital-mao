# User Personas - Digital Mao

## Persona 1: The Card Game Enthusiast

### Demographics
- **Name:** Sarah Chen
- **Age:** 28
- **Occupation:** Software Developer
- **Gaming Experience:** Plays card games regularly (Poker, Hearts, Uno)
- **Tech Savviness:** High - comfortable with web applications

### Goals
- Discover new and challenging card games
- Play with friends remotely during evenings
- Experience novel game mechanics
- Master complex rule systems

### Pain Points
- Bored with traditional card games that have static rules
- Difficulty coordinating in-person game nights
- Wants intellectual challenge beyond luck-based games
- Frustrated by games that don't reward strategic thinking

### Motivations
- Enjoys the "aha!" moment of discovering patterns
- Competitive nature drives her to master new games
- Values social interaction during gameplay
- Appreciates games that reward observation and memory

### Behaviors
- Plays online games 3-4 times per week
- Active in gaming communities and Discord servers
- Reads strategy guides and watches gameplay videos
- Prefers games with depth and replayability

### Needs from Digital Mao
- Clear visual feedback when actions succeed or fail
- Ability to chat with other players during games
- Fast-paced gameplay without long waiting periods
- Fair enforcement of rules (even if rules are hidden)

### Quote
> "I love games that make me think differently. The mystery of discovering hidden rules sounds incredibly engaging!"

## Persona 2: The Mao Veteran

### Demographics
- **Name:** Marcus Rodriguez
- **Age:** 35
- **Occupation:** High School Teacher
- **Gaming Experience:** Played Mao extensively in college
- **Tech Savviness:** Medium - uses apps but not a power user

### Goals
- Relive the fun of college Mao games
- Introduce Mao to new players online
- Play with geographically distributed friends
- Create clever new rules that challenge players

### Pain Points
- Can't organize in-person Mao games anymore
- Friends have moved to different cities
- Misses the social aspect of physical card games
- Worried digital version won't capture Mao's essence

### Motivations
- Nostalgia for college gaming sessions
- Enjoys teaching and watching others learn
- Loves the creative aspect of rule creation
- Values the social bonding through shared confusion

### Behaviors
- Occasionally plays online card games
- Stays in touch with college friends via video calls
- Enjoys games with social deduction elements
- Appreciates games that create memorable moments

### Needs from Digital Mao
- Authentic Mao experience (cryptic penalties, no explanations)
- Easy way to invite specific friends to games
- Ability to create nuanced rules in natural language
- Chat system that captures the banter of physical games

### Quote
> "Mao was the best part of game nights in college. I hope this captures that magic of everyone being confused together."

## Persona 3: The Curious Newcomer

### Demographics
- **Name:** Jamie Park
- **Age:** 22
- **Occupation:** College Student (Psychology Major)
- **Gaming Experience:** Casual mobile games, some board games
- **Tech Savviness:** High - digital native

### Goals
- Learn about Mao after hearing friends talk about it
- Understand the game without explicit instructions
- Have fun even while making mistakes
- Eventually master the hidden rules

### Pain Points
- Intimidated by games with steep learning curves
- Frustrated by unclear feedback
- Worried about looking foolish in front of experienced players
- Concerned about being penalized unfairly

### Motivations
- Curiosity about unique game mechanics
- Enjoys puzzle-solving and pattern recognition
- Wants to be part of the "in-group" who knows Mao
- Appreciates games that teach through experience

### Behaviors
- Watches tutorial videos before trying new games
- Prefers low-stakes environments for learning
- Asks questions in chat when confused
- Learns best through trial and error with guidance

### Needs from Digital Mao
- Gentle introduction without overwhelming penalties
- Ability to observe games before playing
- Helpful (but cryptic) feedback from the judge
- Patient community of players
- Clear indication of when they've broken a rule (even if not why)

### Quote
> "I've heard Mao is this mysterious card game where you learn by failing. That sounds frustrating but also really intriguing!"

## Persona 4: The Rule Creator

### Demographics
- **Name:** Alex Thompson
- **Age:** 31
- **Occupation:** Game Designer
- **Gaming Experience:** Extensive - designs tabletop and digital games
- **Tech Savviness:** Very High - understands game systems deeply

### Goals
- Win games to earn rule creation privileges
- Design creative and balanced new rules
- See how players adapt to their rules
- Push the boundaries of the rules engine

### Pain Points
- Frustrated when rule ideas are too complex to implement
- Disappointed if rules don't create intended gameplay effects
- Worried about breaking game balance
- Wants to see how rules interact with existing ones

### Motivations
- Creative expression through game design
- Enjoys experimenting with game mechanics
- Wants to create memorable gameplay moments
- Interested in emergent complexity from simple rules

### Behaviors
- Analyzes game systems and looks for edge cases
- Thinks strategically about rule interactions
- Writes detailed rule descriptions
- Tests ideas mentally before proposing them

### Needs from Digital Mao
- Natural language rule input that's expressive
- Feedback on rule feasibility and conflicts
- Ability to see how rules are interpreted
- Confirmation that rules work as intended
- Statistics on rule enforcement and violations

### Quote
> "The idea of creating rules in natural language that an AI compiles is fascinating. I want to see how far I can push the system!"

- **Gaming Experience:** Moderate - plays party games and social games

## Design Implications

### For All Personas
- Clear visual feedback for game actions
- Real-time chat integration
- Fair and consistent rule enforcement
- Responsive, intuitive interface

### Specific Considerations
- **Enthusiasts:** Advanced statistics, rule discovery tracking
- **Veterans:** Authentic Mao experience, easy friend invites
- **Newcomers:** Gentle learning curve, observer mode
- **Creators:** Expressive rule language, validation feedback
- **Social Players:** Private games, emoji/reactions in chat
