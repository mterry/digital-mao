# Empathy Maps - Digital Mao

## Empathy Map 1: The Curious Newcomer (Jamie Park)

### SAYS
- "I've never played Mao before, but it sounds interesting"
- "Wait, what did I do wrong?"
- "Can someone explain the rules?"
- "I'm getting penalized but I don't know why"
- "This is confusing but kind of fun"
- "I think I'm starting to see a pattern..."

### THINKS
- *I hope I don't look stupid in front of other players*
- *There must be a pattern to these penalties*
- *Maybe if I watch what others do, I'll learn faster*
- *I wish someone would just tell me the rules*
- *This is frustrating but I want to figure it out*
- *I'm starting to understand some of the rules!*

### DOES
- Observes other players' actions carefully
- Tries different card plays to test theories
- Asks questions in chat (even if they go unanswered)
- Takes notes about what causes penalties
- Mimics successful plays from other players
- Experiments cautiously with new actions
- Reads penalty messages multiple times for clues

### FEELS
- **Confused** when first joining a game
- **Frustrated** by cryptic penalty messages
- **Curious** about the hidden rules
- **Anxious** about making mistakes
- **Excited** when discovering a rule
- **Proud** when successfully playing without penalties
- **Engaged** by the mystery and challenge

### PAINS
- Steep learning curve with no explicit instructions
- Fear of being judged by experienced players
- Uncertainty about whether they're improving
- Feeling lost when multiple rules are active
- Worry about slowing down the game for others

### GAINS
- Satisfaction from discovering rules independently
- Sense of accomplishment when mastering mechanics
- Belonging to the "in-group" who understands Mao
- Improved pattern recognition skills
- Fun stories about learning experiences
- Confidence boost from overcoming challenges

## Cross-Persona Insights

### Common Emotional Journey
1. **Initial Confusion** → All personas experience uncertainty at first
2. **Growing Curiosity** → Mystery drives engagement
3. **Breakthrough Moments** → Rule discovery creates satisfaction
4. **Mastery & Creativity** → Advanced players focus on rule creation

### Universal Needs
- **Clear Feedback:** Even cryptic messages must be consistent
- **Fair Enforcement:** AI judge must be reliable and predictable
- **Social Connection:** Chat and community are essential
- **Progressive Challenge:** Game must scale from beginner to expert

### Key Touchpoints
- **First Penalty:** Critical moment that sets tone
- **First Rule Discovery:** Validates the learning approach
- **First Win:** Unlocks rule creation capability
- **First Rule Creation:** Transforms player into contributor

### Design Priorities
1. **Onboarding:** Help newcomers without breaking mystery
2. **Feedback Quality:** Cryptic but informative penalties
3. **Social Features:** Enable community and teaching
4. **Rule Creation:** Expressive yet constrained system
5. **Progression:** Clear sense of improvement over time

---

## Emotional Arc Across User Journey

```
Confusion → Frustration → Curiosity → Discovery → 
Satisfaction → Mastery → Creativity → Teaching
```

### Critical Success Factors
- First 5 minutes must hook players despite confusion
- Penalties must feel fair even when mysterious
- Discovery moments must feel earned and rewarding
- Rule creation must feel powerful but balanced
- Community must be welcoming to newcomers
